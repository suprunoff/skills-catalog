## Key Findings

### 1. **The Anthropic Dominance**
- **Single repo (anthropics/skills) = 94,961 stars** — nearly 50% of the entire top 20 combined
- 10,192 forks indicate widespread integration
- **No competition at this level** — next repo (obra/superpowers) at 88,226 stars is a framework, not a skills collection

### 2. **Multi-Platform Ecosystem**
No single skill standard yet. Repos serve different platforms:
- **Claude Code** (most popular)
- **OpenClaw** (5,400+ skills, growing fast)
- **Cursor** (increasingly popular)
- **Copilot** (GitHub official)
- **Gemini** (limited)

### 3. **Community-Driven Growth**
- VoltAgent/awesome-openclaw-skills (38,319 stars) became #2 collection within 1-2 years
- Shows OpenClaw ecosystem maturity
- Community curates what Anthropic doesn't officially maintain

### 4. **Two Business Models Emerge**
- **Official repos** (Anthropic, Vercel, Microsoft) — maintained, quality-gated, slow updates
- **Community collections** — fast-growing, 1000-5400+ skills, but quality varies

### 5. **Specialization Matters**
- Single-purpose skills with clear value (humanizer: 9,531 stars, planning-with-files: 16,207 stars) outperform generic collections
- **Lesson:** specificity drives adoption more than breadth

### 6. **OpenClaw is Now Mainstream**
- 3 major repos dedicated to OpenClaw (rank #3, #13 cross-platform collection)
- VoltAgent/awesome-openclaw-skills rivals Vercel-labs/agent-skills in adoption
- OpenClaw skillset now competitive with Claude Code ecosystem

## Competitive Landscape

### Strengths by Repo Type

| Type | Strength | Example |
|------|----------|---------|
| **Official** | Trust, maintenance, integration | Anthropic, Vercel, Microsoft |
| **Collections** | Speed, breadth, community input | VoltAgent awesome-openclaw-skills |
| **Frameworks** | Complete system, methodology | Superpowers, Wasp |
| **Specialized** | Deep focus, high adoption | Humanizer (9.5K stars) |

### Threats
- **Fragmentation:** Multiple incompatible skill formats
- **Quality variance:** Community repos lack QA
- **Vendor lock-in:** Each platform owns their ecosystem
- **Skill decay:** Community skills become outdated (no deprecation policy)

### Opportunities
1. **Skill marketplace with verification** (like Vercel's recent move)
2. **Cross-platform skill transpiler** (Claude Code → OpenClaw → Cursor)
3. **Skill versioning system** (currently ad-hoc)
4. **Enterprise skill management** (governance, approval workflows)

## Recommendations

### For Skill Developers
1. **OpenClaw is now equal to Claude Code** — develop for both
2. **Specialization > generality** — narrow scope, deep functionality
3. **Fork ratio matters** — if >15% forks, you're hitting customization needs
4. **Maintenance is critical** — repos not updated in 2+ weeks lose visibility

### For Platform Builders
1. **Invest in skill verification** — quality is the next differentiator
2. **Cross-platform compatibility layer** — users want one skill everywhere
3. **Skill marketplace + curation** — community-driven growth (proven by VoltAgent)
4. **Developer experience tools** — skill CLI (like Vercel's) accelerates adoption

### For AI Teams
1. **Start with anthropics/skills** — 94K stars, most integrated, best docs
2. **Layer with Vercel-labs/agent-skills** — production-grade official support
3. **Use VoltAgent awesome-openclaw-skills** — if OpenClaw target
4. **Watch emerging single-skill patterns** — humanizer at 9.5K stars shows niche wins
