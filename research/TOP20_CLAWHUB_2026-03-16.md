## Findings & Analysis

### Download Volume Leaders
1. **Capability Evolver** (35.6K) is THE dominant skill—more than 2x any peer
   - Clear use case: agents self-improving through session log analysis
   - High adoption suggests this is near-essential for advanced agent workflows
   
2. **Wacli/ByteRover/ATXP** (14-16K each) form a utility cluster
   - Practical day-to-day automation tools
   - Lower star counts suggest broader appeal but less "wow factor"

3. **Self-Improving Agent** (15.9K downloads, 132 stars) is the quality champion
   - Nearly equal downloads to Wacli but 3.6x more stars
   - Indicates better documentation, UX, or real-world impact

### Category Distribution (Top 20)
- **Utility/Automation:** 7 skills (Wacli, ByteRover, ATXP, Bird, Find Skills, Nano Banana, Humanize)
- **AI/ML:** 4 skills (Capability Evolver, Self-Improving Agent, Proactive Agent, Free Ride)
- **Development/GitHub:** 3 skills (Gog, GitHub, Tavily Web Search)
- **Productivity/Knowledge:** 3 skills (Summarize, Obsidian, Nano Banana)
- **Web/Automation:** 1 skill (Agent Browser)
- **Data/IoT:** 2 skills (Weather, Sonoscli)

### Security & Quality Signals
- **Certified skills:** Only "Self-Improving Agent" explicitly noted as certified in multiple sources
- **High stars without high downloads:** Proactive Agent (49 stars, 7K downloads) suggests smaller but dedicated user base
- **Red flag research:** Snyk reported [280+ skills with credential leaks](https://snyk.io/blog/openclaw-skills-credential-leaks-research/) — ~20% of platform poses security risk

### Ecosystem Health
- **Total registered skills:** 13,286 (Feb 2026)
- **Total downloads:** 1.5M+ cumulative
- **Active installations:** 6,375 (from TopClawHubSkills tracking)
- **Growth:** +3.7K downloads/week (Tavily Web Search alone in week of Mar 1-2)

## Key Recommendations

### For New Users
1. **Start with Big 3:** Capability Evolver + Self-Improving Agent + Gog cover most use cases
2. **Verify source:** Check author, code quality, and permission scopes before install
3. **Monitor VirusTotal:** ClawHub now runs VirusTotal scans post-ClawHavoc incident (2,419 suspicious skills removed)

### For Agent Builders
- **Self-evolution:** Capability Evolver + Self-Improving Agent are almost required for production agents
- **Integration backbone:** Gog + GitHub cover most enterprise workflows
- **Research-heavy:** Tavily Web Search outperforms generic web search

### Security Concerns
- ~20% of ClawHub skills contain unvetted API keys or permission requests
- Recommendation: Use certified-only filter when available
- Post-install: Audit `SKILL.md` for prompt injection vectors and hardcoded credentials
