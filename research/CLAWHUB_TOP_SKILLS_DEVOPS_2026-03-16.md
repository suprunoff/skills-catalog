## Security Scanning Status

### Pre-Install Protocol (MANDATORY)

**Before installing ANY skill:**
1. Visit `https://clawhub.ai/[author]/[skill-name]`
2. Check VirusTotal report (bottom of page, OpenClaw partnership)
3. Run `skill-vetter` locally before activation
4. For high-risk skills: human code review (README + SKILL.md files)

### Known Risks (2026)

- **ClawHavoc campaign** (Jan 27-29, 2026): 341 malicious skills in 5,400+ audited
- **ClawHub remediation:** Removed 2,419 suspicious skills; partnered with VirusTotal
- **Prompt injection:** 36% of published skills contain prompt injection vulnerabilities (Snyk)
- **False negatives:** Skill Vetter catches obvious red flags but NOT cleverly obfuscated/time-delayed malware

### Verification Tools

| Tool | Purpose | Trust |
|------|---------|-------|
| **VirusTotal (OpenClaw partnership)** | Automatic malware scan | ✓ Official |
| **Skill Vetter** | Pre-install checklist | ✓ 97/100 score |
| **Bitdefender AI Skills Checker** | Third-party scan | ✓ Independent |
| **Human code review** | For HIGH-risk auth/payment skills | ✓ Required for prod |

## Architecture Reference

### Skill Format (Portable)

All ClawHub skills follow a standard SKILL.md format:
- YAML frontmatter: metadata, author, version
- Natural-language instructions (no code)
- Compatible with Claude Code, Cursor, OpenClaw

### Integration

Skills connect via:
- **MCP (Model Context Protocol)** — structured APIs
- **CLI wrappers** — `gh`, `gog`, Playwright
- **Auth flows** — GitHub OAuth, API keys (env vars)
- **File system** — read/write with permission gates
